Allow players to choose from multiple character classes with distinct passive abilities before queuing into a round.

Depends on: player-movement, skill-tree, main-menu

- At least two character classes available at launch (e.g. Assault, Scout)
- Each class has a unique passive ability (e.g. Assault: +15% weapon damage; Scout: +20% move speed)
- Character definitions in `data/characters.json` with stats and ability description
- Selection screen accessible from main menu before starting a round
- Selected class is saved in save data and persists between sessions
- Skill tree nodes can be class-gated (optional: some nodes only available to certain classes)
